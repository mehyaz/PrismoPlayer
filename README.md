# PrismoPlayer

Premium video player with intelligent content filtering.

## Features

*   **Premium Video Player:** Enjoy a seamless video playback experience.
*   **Intelligent Content Filtering:** Skip unnecessary scenes with our smart filter.
*   **Torrent Streaming:** Stream videos directly from torrents.
*   **Cross-Platform:** Available for Windows, macOS, and Linux.

## Technologies Used

*   **Electron:** For building the desktop application.
*   **React:** For the user interface.
*   **TypeScript:** For type safety.
*   **Tailwind CSS:** For styling.
*   **Vite:** For build tooling.
*   **WebTorrent:** For streaming torrents.
*   **Cheerio:** For web scraping.

## How to Use

1.  **Installation:**
    *   Clone the repository: `git clone https://github.com/your-username/PrismoPlayer.git`
    *   Install dependencies: `npm install`
2.  **Development:**
    *   Run the application in development mode: `npm run dev`
3.  **Production:**
    *   Build the application: `npm run build`
    *   The distributable files will be in the `dist` folder.
